# Aponte - Expo Router (SDK 51) TSX

Projeto mínimo com Expo Router, TypeScript (TSX) e StyleSheet.  
Contém navegação stack + tabs + modais. Manual dark-mode toggle presente.

Como executar:
1. Extraia o projeto.
2. Rode `npm install` dentro da pasta do projeto.
3. Rode `npx expo start` (ou `npm start`).
